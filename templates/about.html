Fraud Detection Methods

Rule-Based Detection (8 Indicators)

  1. Impossible Travel (30%) - Detects physically impossible transaction distances
  2. Rapid Transactions (25%) - Flags multiple transactions in short timeframes
  3. Amount Anomaly (20%) - Uses Z-score analysis for unusual amounts
  4. Country Shift (20%) - Identifies new geographic locations
  5. High Frequency Day (15%) - Detects unusual daily transaction volume
  6. New Merchant (15%) - Identifies transactions with unknown merchants
  7. Time Anomaly (10%) - Detects unusual transaction times
  8. Category Deviation (10%) - Identifies unusual merchant categories

Machine Learning Detection

In addition to rule-based detection, the system includes trained ML models that can learn patterns from historical data:

  • Logistic Regression - Fast linear model for probability prediction
  • Random Forest - Ensemble method for robust predictions
  • Ensemble Voting - Combines both models for improved accuracy
